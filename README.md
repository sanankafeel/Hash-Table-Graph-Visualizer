# 🗂️ Hash Table & Graph Visualizer

An interactive desktop application built in Python that demonstrates the working principles of **Hash Tables** and **Graphs** through an intuitive graphical user interface (GUI). The project helps users understand fundamental data structures by providing real-time visualization of their operations.

## ✨ Features

### Hash Table
- Insert key-value pairs
- Search for existing keys
- Delete entries
- Collision handling visualization
- Dynamic table updates

### Graph
- Create and modify graphs
- Add/remove vertices and edges
- Visual graph representation
- Interactive graph traversal
- Educational visualization of graph operations

## 🛠️ Technologies Used

- Python
- Tkinter
- Object-Oriented Programming (OOP)
- Data Structures & Algorithms
